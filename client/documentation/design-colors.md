## 2. Color Application Standards (How to Use Them)

### A. Primary Navy (`#002D62` - `#191970`) - **70% of Color**

This color should form the **foundation** of your branding and text for maximum readability.

* **Header and Footer:** Solid background for the main navigation bar and website footer.
* **Primary Text:** All main body text, headings (`H1`, `H2`, `H3`), and labels should use this color (or black, if better for contrast).
* **Primary Buttons:** Used as the background for non-CTA buttons (e.g., "Learn More," "Settings").
* **Icons:** Used for all non-accent interface icons.

### B. Accent Teal (`#00FFD8` - `#39FF14`) - **20% of Color**

This color is for **action and emphasis**. Use it sparingly to guide the user's eye.

* **Call-to-Action (CTA) Buttons:** The background color for critical actions like **"Scan Receipt Now"** or **"Sign Up."**
* **Links and Hyperlinks:** Color for all inline text links.
* **Status Indicators:** Success messages, 'Active' status badges, and progress bars.
* **Highlights:** Key elements of the logo and visual graphics.
* **Gradients:** Use the **Navy-to-Teal gradient** for large hero banners or prominent graphical elements, mimicking the logo's dynamic style.

### C. Neutral White (`#FFFFFF`) - **90% of Background**

White ensures **readability and cleanliness**.

* **Main Background:** The primary background color for all content areas and pages.
* **Cards and Modules:** Background color for content cards, data tables, and input fields to make them stand out.
* **Text on Dark Areas:** Used for all text/icons placed over the **Primary Navy** (e.g., in the header/footer).

### D. Neutral Grey (`#D3D3D3` - `#EAEAEA`) - **10% of Support**

This provides **structure and subtlety**.

* **Dividers:** Horizontal rules and subtle section breaks.
* **Borders:** Outlines for input fields, content cards, and non-critical components.
* **Shadowing:** Use the light grey for subtle box shadows to create depth (mimicking the logo's 3D circle effect).
* **Secondary Text/Labels:** Used for descriptive text, helper text, or disabled element text to reduce visual noise.

---

## 3. Design Standards Compliance

* **Contrast:** Always ensure **WCAG AA contrast standards** are met, especially between **Navy Text** and **White Background**, and **Teal Accent** and **White Background**.
* **Consistency:** The **Accent Teal** should *only* be used for interactive elements (CTAs) to avoid confusing the user about where to click.
* **Logo Gradient:** The specific Navy-to-Teal gradient should be reserved for the **logo** and limited brand assets to maintain its visual impact.
